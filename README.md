# 🎉 생일 카페의 모든 것, 버카 🎉

버카는 생일 카페를 위한 통합 플랫폼입니다. 사용자들은 '방문자', '주최자', '사장님' 세 가지 주체로 나뉘어 각자의 역할을 수행합니다. 

## 📱 앱 다운로드
버카 앱을 통해 생일 카페의 모든 서비스를 간편하게 경험해보세요.  
[앱 다운로드 링크](https://apps.apple.com/kr/app/%EB%B2%84%EC%B9%B4/id6504910184)
